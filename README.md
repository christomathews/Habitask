# Habitask - Elevate Your Routine

Habitask is a modern, minimalist habit-tracking application designed to help you build better routines through consistency and data-driven insights. 

![Habitask Preview](https://via.placeholder.com/800x450?text=Habitask+Interface+Preview) <!-- Placeholder for a real screenshot -->

## ✨ Features

- **Personalized Accounts**: Simple, local-first authentication to keep your data private to your browser.
- **Smart Habit Tracking**: Add, delete, and manage your daily habits with a clean, intuitive interface.
- **Precision Time Tracking**: 
  - **Live Timer**: Track habits in real-time as you perform them.
  - **Manual Entry**: Quickly log time spent if you forgot to start the timer.
- **Dynamic Analytics**: 
  - **Completion Ratio**: Visualize your consistency over time (Week/Month/Year).
  - **Time Distribution**: See exactly how much time you're investing in each habit.
- **Date Navigation**: Easily review past performance or prepare for tomorrow.
- **Futuristic UI**: High-contrast dark mode with glassmorphism effects and Lucide icon integration.

## 🚀 Getting Started

Habitask is a lightweight, client-side application. No server setup or database installation is required.

### Prerequisites

All you need is a modern web browser (Chrome, Firefox, Safari, or Edge).

### Installation & Running

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/habitask.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd habitask
   ```
3. **Open the app**:
   Simply open `index.html` in your browser, or use a local development server like Live Server (VS Code extension).

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Flexbox, Grid, and Glassmorphism.
- **Vanilla JavaScript**: State management and core logic (No heavy frameworks!).
- **[Lucide Icons](https://lucide.dev/)**: For beautiful, consistent iconography.
- **LocalStorage API**: For persistent data storage without a backend.
- **SVG**: For custom, lightweight analytics charts.

## 📖 How to Use

1. **Sign Up**: Create a local account on the splash screen.
2. **Add Habits**: Use the input bar at the top to list the routines you want to build.
3. **Track**: 
   - Click a habit to open the **Timer Modal**.
   - Use the **Start/Stop** buttons or enter minutes manually.
   - Use the **Complete** button for simple check-offs.
4. **Analyze**: Switch to the **Analytics** tab to view your performance charts.
